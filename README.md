
# NeoCore – Under Construction

Proyecto Next.js mínimo para dejar el dominio **estable** mientras trabajamos en NeoCore.

## Desarrollo

```bash
npm i
npm run dev
```

## Deploy rápido en Vercel

1. Sube este repo a GitHub (nuevo repositorio vacío).
2. En Vercel: *New Project* → Importa el repo → Deploy.
3. (Opcional) Añade tu dominio y prueba `/api/ping`.

Sin CSP ni middleware agresivo. Trae una página simple y un endpoint de salud.
